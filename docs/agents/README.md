# Agent playbooks

Agents operate under root `AGENTS.md`, `agent-policy.json`, and the shared permission policy. Choose the narrowest owning agent; the Orchestrator coordinates cross-domain tasks. Required handoffs use:

```text
Summary:
Scope/files:
Architecture reused:
Security/tenant impact:
Migrations/integrations:
Checks and results:
Approvals/reviews:
Rollback/forward recovery:
Open risks/limitations:
Draft PR:
```

Role documents: Orchestrator, Backend, Frontend, Database, Security, QA, DevOps, Etsy Integration, Billing, AI Design, Documentation, and Release Manager.
